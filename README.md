# U.S. Data Analyst Job Market Analysis

## Overview
An Excel dashboard analyzing 7,800+ U.S. data analyst job postings 
from 2020–2025.

## Key Findings
- Entry-level data analysts average $94K in the U.S.
- Salaries peaked in 2022 at $115K
- Post-2022 decline likely tied to tech market corrections

## Tools Used
- Microsoft Excel
- Pivot Tables
- PivotCharts

## Data Source
Kaggle - Latest Data Science Job Salaries 2020-2025
